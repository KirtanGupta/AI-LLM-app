# Simple AI LLM Web App

A basic web application built with Node.js and Groq API.

## Features
- User prompt input
- AI-generated response
- Backend API integration
- Secure environment variable usage

## Tech Stack
- Node.js
- Express.js
- Groq API
- HTML + JavaScript

## Setup Instructions

1. Clone the repository
2. Run: npm install
3. Create a .env file
4. Add:
   GROQ_API_KEY=your_key_here
5. Run:
   node server.js
6. Open:
   http://localhost:5000
